## 如何使用

1. 安装插件
2. 打开「DutyIsland 值日表」设置界面
3. 设置值日表模板
4. 设置值日表
5. 打开「编辑档案」
6. 选择课表，点击「详细设置」
7. 滑到最下面找到「高级设置」->「值日表设置」
8. 启用并设置值日表

完成以上步骤后再设置组件、自动化，因为这些东西的下拉框内容是从当前启用的课表中的值日表高级设置读取的。
