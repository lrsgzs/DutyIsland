## 📝 附记

本次 release 包含 0.2.2.0 的更新日志。

## 🚀 新增功能与优化

- 【设置】拆分设置界面，支持 ClassIsland 2.0.1.0 的设置界面分组
- 【轮换】在左侧突出当前索引
- 【点名】添加了 SecRandom 广告位
- 【档案】优化执行者搜索，支持拼音模糊匹配
- 【主页】在主页显示 README 内容

## 🐛 Bug 修复

- 【轮换】修复偏移错误、无法轮换的问题
- 【联动】修复插件联动时，未检查目标插件是否启用，导致 ClassIsland 服务出问题

## 🪛 技术性更改

- 【事务】更新 PluginSdk 到 2.0.0.2
- 【核心】分离出 DutyIsland.Interface
- 【核心】移除自定义 Logger 实现，换成 Microsoft.Extension.Logging
    - 本来这个是有强行拉取 AppLoggerProvider 的，可以实现在 ClassIsland
      主机启动前向应用的日志界面添加日志，觉得不稳当就注释掉了。
- 【核心】动态注入修改属性，实现只在高版本适配设置界面分组
- 【事务】现在好像可以自动构建了？
